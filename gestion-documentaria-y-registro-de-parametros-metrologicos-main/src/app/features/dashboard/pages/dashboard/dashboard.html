Guía de Componente: app-btn

Sistema de botones estandarizado con Angular Material + Phosphor Icons.

Nota de alcance:

Los componentes mostrados aquí son herramientas de desarrollo externas. No están integrados en el flujo operativo del sistema final.

Matriz de Variantes

Variante Uso Sugerido Concepto Visual
primary Guardar, Confirmar, Nuevo.
Guardar
outline Editar, Ver más, Exportar.
Exportar
soft Filtros activos, Selección.
Seleccionar
secondary Limpiar, Cancelar, Volver.
Neutro

Implementación Estándar 

<app-btn
  variant="primary"
  icon="ph-floppy-disk"
  (click)="save()">
  Guardar Usuario
</app-btn>

Solo Icono + Tooltip 

<app-btn
  variant="soft"
  [onlyIcon]="true"
  icon="ph-pencil"
  tooltip="Editar registro">
</app-btn>

Importante para Iconos:

Usamos Phosphor Icons. Busca los nombres en phosphoricons.com. Pasa solo el nombre (ej: ph-user) al input icon.
